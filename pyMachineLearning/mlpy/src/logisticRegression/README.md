## 逻辑回归logistic regression
逻辑回归本质上是针对已有的一批数据，拟合一个多项式的权重向量，来使得拟合误差最小  
如果采用原始的精确算法，即每一次梯度下降对全量数据进行组合叠加，那么大数据的计算时间将会不可接受（M*N,如果M很大N适中，那意味着很庞大的计算量）  
采用随机梯度下降法，在效率和精度上做了权衡，即在可控的迭代周期内，每次震荡衰减alpha步长，并且，只选取单条样本数据进行误差的计算和修正叠加  
这样就可以在很低的维度内对数据进行逻辑回归
